Installation

Install EPI Recorder from PyPI in seconds. No complex setup required - just one simple command.

# Install from PyPI pip install epi-recorder # Verify installation epi --version

  • Works on Windows, macOS, Linux
  • All dependencies auto-installed
  • Ready to use immediately

Record with EPI

Run your script with EPI to create a cryptographically signed recording of the entire execution.

epi run complete_demo_workflow.py \ --goal "Text summarization demo" \ --notes "Investor presentation" # Alternative: CLI recording epi record --out demo.epi -- \ python workflow.py
  • Captures all inputs/outputs
  • Records execution environment
  • Auto-redacts API keys & secrets
  • Cryptographically signed (Ed25519)
  • Creates single .epi file

Verify Authenticity

Verify the recording's integrity and authenticity using cryptographic verification.

epi verify complete_demo_workflow.epi # Output shows three-level verification: # 1. Structural validation # 2. Integrity check (SHA-256) # 3. Signature verification (Ed25519)
  • Tamper-proof verification
  • Cryptographic proof of authenticity
  • Trust level: LOW/MEDIUM/HIGH
  • Detects any modifications
$ epi verify complete_demo_workflow.epi +-------------- [OK] EPI Verification Report --------------+ | File: complete_demo_workflow.epi | | Trust Level: HIGH | | Message: Cryptographically verified and integrity intact | | | | [OK] Integrity: Verified (4 files) | | [OK] Signature: Valid (key: default) | +----------------------------------------------------------+ Verification Details: ✓ ZIP structure valid ✓ Manifest schema valid ✓ All file hashes match (SHA-256) ✓ Ed25519 signature valid ✓ No tampering detected This recording is authentic and unmodified.
